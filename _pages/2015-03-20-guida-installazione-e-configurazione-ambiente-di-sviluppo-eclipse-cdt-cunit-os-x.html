Installazione Eclipse CDT in OS X

Per installare Eclipse CDT su sistemi Mac OS X,  occorre aver installato X Code, la suite di sviluppo di Apple, che porta con sé il compilatore C. Una volta installato, anche Eclipse CDT funzionerà correttamente.

Per maggiori dettagli, consultare le guide alle seguenti pagine:

  • http://max.berger.name/howto/cdt
  • http://ntraft.com/installing-gdb-on-os-x-mavericks
  • http://macappstore.org/cunit
